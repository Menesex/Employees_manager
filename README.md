# 🚀 Mene | Employee Management System

Sistema profesional de gestión de empleados diseñado para el control administrativo de personal y áreas. Proyecto desarrollado con un enfoque en arquitectura de nube escalable y seguridad de datos.

👉 [Mene-HR Live App](https://employee-management-system-fmju.onrender.com)

---

## 💡 Sobre el Proyecto
Este es un **proyecto educativo** integral que demuestra la integración de múltiples capas de software (Full Stack) y el despliegue de infraestructura distribuida.

### 🛠️ Stack Tecnológico
* **Backend:** FastAPI (Python 3.9+) 🐍
* **Base de Datos:** PostgreSQL (Hosteado en Supabase) 🐘
* **Frontend:** Vanilla JavaScript, HTML5 y CSS3 (UI Moderna) 🎨
* **ORM:** SQLAlchemy (Abstracción de base de datos) ⚙️
* **Despliegue:** Render.com (CI/CD Automático) ☁️

## 🌟 Características Principales
* **Gestión CRUD:** Registro, edición y retiro de empleados en tiempo real.
* **Persistencia Universal:** Base de datos compartida en la nube (todos los usuarios ven los mismos datos).
* **Soft-Delete:** Historial de empleados retirados sin pérdida de datos.
* **Seguridad:** Validación de datos con Pydantic y protección contra ataques XSS.
* **Filtros Inteligentes:** Búsqueda dinámica por nombre, ID o tarjeta de acceso.
